# Данные, Величины, Типы, Операторы

## Величины в JS - блоки информации.

## Основные типы величин:        
#### числа

целые(5), с точкой(5.25), специальные(NaN, Infinity, -Infinity)

#### строки

 "", '', экранирование (\) и спец символы (\t, \n)
 
#### булевые

false, true

#### неопределённые
 
null, undefined 

#### обьекты

{}

#### функции

function

## Основные операторы:
   
####Арифметические

**+ - * / %**

```javascript
5 + 8;
13 - 21;
((8 * 21) - 55 / 5) + 89;
10 / 0;
0 / 0;
```

```javascript
"привет" + " " + "человек";
"это первая строка,/nа это вторая";
```

#### Сравнения

**== === != !== > < >= <=**

```javascript
5 > 8;
3 < 13;
21 == 21;
34 != 34;
"hello" == "hello";
"Hello" == "hello";
```

#### Логические

&& || ! ?:

```javascript
false || true;
false && true;
true && true;
true ? 13 : 21;
false ? 13 : 21
21 > 15 ? "21 больше 15" : "21 меньше 15";
```

#### Определение типа

typeof  

```javascript
typeof 5;
typeof "hello";
typeof true;
typeof {};
typeof undefined;
```
